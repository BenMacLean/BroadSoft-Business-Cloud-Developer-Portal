Micro App Help

The micro app can be seen as the main portal that the user will see. This is also where the user will navigate to when they enable your application in the settings view.

Let's start by enabling your application from the settings page.

What we will send you:

GET http://yourUrl/v1/yourAppName/authenticate?id=username

Then you send us:

ResponseCode: 200 { auth: 'your custom auth token' }
